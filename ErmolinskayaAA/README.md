# Практическая работа №104: Docker

## Выполнила: Ермолинская А.А.

## Цель работы
Познакомиться с Docker, изучить базовые команды Linux, научиться создавать и управлять Docker-контейнерами, публиковать образы на Docker Hub.

## Задачи
1. Создать и запустить контейнер на основе Alpine Linux
2. Установить bash и выполнить все действия внутри контейнера
3. Выполнить 13 заданий по работе с файлами, каталогами, текстом и датой
